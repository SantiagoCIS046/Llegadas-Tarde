# Backend - LlegadasTarde

Esta carpeta está destinada para el código del backend del proyecto LlegadasTarde.

## Estructura sugerida

Aquí puedes desarrollar tu API backend utilizando tecnologías como:
- Node.js + Express
- Python + Flask/Django
- PHP + Laravel
- Java + Spring Boot
- Etc.

## Próximos pasos

1. Inicializar tu proyecto backend
2. Configurar la base de datos
3. Crear las rutas y controladores necesarios
4. Implementar la lógica de negocio
